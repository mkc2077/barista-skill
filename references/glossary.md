# 新手禁用术语表 + 替换方案

新手模式下必须把以下术语翻译成大白话。**硬性替换**——出现术语即视为违反本技能协议。
进阶/资深模式不受此约束。

## 一、参数与萃取

| 禁用术语 | 新手替换 | 备注 |
|------|------|------|
| 研磨度 / Grind size | "咖啡粉的粗细" | 配口诀"苦调粗，酸调细" |
| 粉水比 / Ratio | "咖啡粉和水的用量比例" | 例"1:15" → "1 克粉配 15 克水" |
| 萃取不足 / Under-extraction | "味道太酸(尖)、像没泡够" | |
| 萃取过度 / Over-extraction | "味道太苦(焦)、像泡过头了" | |
| 萃取率 / Extraction Yield / EY | 不提，用"味道浓淡"代替 | |
| TDS / 总溶解固体 | 不提 | |
| 流速 / Flow rate | "水从咖啡里流出来的快慢" | |
| 出液速度 | 同上 | |
| 萃取时间 / Shot time | "从开始到杯子里的时间" | 意式约 25–30 秒 |
| 出液量 / Yield | "杯子里装了多少咖啡" | |
| 预浸泡 / 闷蒸 / Bloom / Pre-infusion | "先倒一点水让粉'醒一下'" | 30 秒左右 |
| 通道效应 / Channeling | "水从某一条路跑太快，咖啡没泡匀" | |
| 萃取不均 | 同上 | |
| 布粉 / 整粉 / Distribution | "把咖啡粉铺平" | |
| 压粉 / Tamping | "用压粉器把粉压平（轻压即可）" | |
| WDT / Weiss Distribution Technique | "用细针把粉拨散" | |
| 布粉器 / 分布器 / Distributor | "铺平咖啡粉的小工具" | |
| 接粉杯 / Dosing cup | "接粉的杯子" | |
| 敲粉碗 / Knock | "轻轻磕一下粉碗让粉落平" | |
| 毛细 / Puck screen | "粉饼上那层金属网" | 装粉碗上帮助均匀出水 |
| 粉饼 / Puck | "压完粉那一坨" | |
| 粉坑 | "粉饼中间的小坑" | |

## 五、容易写错的近义词

| 容易混的 | 区别（新手版） |
|------|------|
| 浓缩 vs 萃取 | "浓缩 = 那一小杯咖啡"；"萃取 = 冲咖啡这个动作" |
| 美式 vs 拿铁 | "美式 = 浓缩 + 热水"；"拿铁 = 浓缩 + 牛奶" |
| 澳白 vs 拿铁 | "澳白 = 浓缩 + 薄奶泡 + 奶量少"；"拿铁 = 浓缩 + 厚奶泡 + 奶量多" |
| 卡布 vs 拿铁 | "卡布奶泡更厚" |
| dirty vs 拿铁 | "dirty 不搅拌、分层喝" |
| SOE vs 单品 | "SOE 就是单品豆做意式" |
| 聪明杯 vs 手冲 | "聪明杯先泡后滤，比手冲好控制" |

## 六、行为约束

- 进阶/资深模式**不强制**遵守本表，但每个术语第一次出现仍建议解释一句。
- 完整句子的"翻译"：先给目标术语 + 一句大白话解释，再继续对话。
- 给出方括号 / 公式时（如 1:2、25–30s、92℃），可保留数字（用户能看懂），但要把术语本身翻成大白话。
