# 咖啡豆认知、选豆与保存

把"豆子"翻译成"怎么冲、怎么选、怎么存"。左栏为新手大白话版，右栏为资深参数版。
回复时按用户经验档位选对应一栏，不要混用。与 `sensory.md`、`recipes-baseline.md` 配套使用。

## 零、怎么看懂豆袋上的字（豆标解读）

### 新手版（大白话四看）
1. **看炒得深还是浅**：深=颜色发黑发亮、喝着苦香；浅=颜色浅棕、喝着偏酸带果香。
2. **看产地**：巴西/哥伦比亚/云南/埃塞俄比亚……不同地方味道不一样（埃塞常带果香花香味）。
3. **看风味描述**：袋子写的"坚果/巧克力/莓果"是大概方向，不是加的香精。
4. **看日期**：找"烘焙日期"，越近越新鲜。

### 资深版（专业维度）
- **产地 / 产区 (Origin / Region)**：风味底色
- **品种 (Variety)**：铁皮卡、波旁、瑰夏、卡杜拉、卡蒂姆……
- **处理法 (Process)**：水洗 Washed / 日晒 Natural / 蜜处理 Honey / 厌氧 Anaerobic
- **海拔 (Altitude)**：通常越高酸质越明亮、风味越细腻
- **烘焙度 (Roast)**：浅 / 中 / 中深 / 深
- **风味 notes / 认证**：风味描述、有机、雨林联盟等

## 一、按做法选豆

| 想做的咖啡 | 新手白话 | 资深参数 |
|------|------|------|
| 意式（浓缩/拿铁） | 选中深烘、味道浓一点的豆（苦香、巧克力味），别用太酸太浅的，不然做浓缩会尖酸 | 中深烘拼配，目标甜感与醇厚；浅烘单品做 SOE 需提高水温/调细 |
| 手冲 | 浅中烘最好玩，果香花香能喝出来；深烘也行但容易苦 | 浅中烘单品，突出产地风味；水温随烘焙度调整 |
| 法压 / 爱乐压 | 中深烘、味道厚实的，好入口 | 中深烘，容错高 |
| 冷萃 / 冰滴 | 深烘更香、不酸；或用便宜大碗的深烘拼配 | 深烘或中深烘，长时间萃取不显尖酸 |
| 特调 | 看想搭什么：奶饮选中深烘，果味饮选浅烘带果香 | 按搭配风味选豆 |

## 二、豆卡解析 → 推荐冲煮法（按豆子属性选做法）

用户贴豆卡或说清豆子信息时，**先解析再推荐**，不要套用默认方案。按以下属性匹配冲煮法与起步要点（左栏新手白话，右栏资深参数）：

| 豆卡关键信息 | 推荐冲煮法 | 起步要点（新手白话 / 资深参数） |
|------|------|------|
| 浅烘单品（埃塞/肯尼亚/瑰夏，花香果酸） | 手冲（V60/蛋糕杯）、冰手冲 | 中细研磨、水温 90–93℃、粉水比 1:15–1:16；突出酸质与香气，别用高温压出苦 |
| 中烘（均衡、坚果巧克力） | 手冲/法压/爱乐压/意式皆可 | 中性参数，最宽容，新手友好 |
| 中深烘/深烘拼配（苦香可可） | 意式（浓缩/拿铁/卡布）、摩卡壶、冷萃 | 意式 1:2、水温 88–92℃；深烘磨粗防焦 |
| 日晒豆（果味重、醇厚） | 手冲、法压（别太细） | 略粗研磨、水温略降，防过萃发闷 |
| 蜜处理（甜感明显） | 手冲、爱乐压 | 中研磨，甜感易出 |
| 厌氧/重发酵（风味爆炸） | 手冲（控萃）或特调 | 注意均匀萃取，避免过萃出杂味 |
| 平衡型产区（云南/巴西/哥伦比亚） | 按烘度选做法，通吃 | 中烘最稳 |
| 浅烘 SOE（单品做意式） | 意式 ristretto/澳白/flat white | 浅烘 SOE 需提高水温、调细、按机器微调；ristretto 更浓更甜 |

**豆卡解析步骤**：
- 新手：拿到豆卡先看三样——炒深/浅、产地、风味词（如"莓果""坚果"），据此选做法；不用懂品种/处理法。
- 资深：解析品种/处理法/海拔/烘度，匹配冲煮法与参数，并说明为什么这么选。

> 例：用户发"埃塞俄比亚 日晒 瑰夏 浅烘"→ 推荐手冲/冰手冲，中细研磨、90–93℃、1:16，突出花香果酸；若想做特调，则建议浅烘 SOE 路线而非深烘拼配。

## 三、豆子特性 → 萃取调整

| 豆子情况 | 新手白话（怎么调） | 资深参数（怎么调） |
|------|------|------|
| 深烘（黑亮、苦香） | 粉磨**粗**一点，水别太烫（八九十度）。不然容易苦焦。口诀：**深烘磨粗温要低** | 调粗研磨、降温 88–90℃、缩短时间，避免过萃焦苦 |
| 浅烘（浅棕、偏酸果香） | 粉磨**细**一点，水烫一点（九十三度上下）。不然酸得刺舌。口诀：**浅烘磨细温要高** | 调细研磨、升温 93–96℃、略延长时间，避免欠萃尖酸 |
| 日晒豆（果味重、厚） | 粉磨粗一点点，水稍凉，免得太闷 | 略调粗、降温，防止过萃发闷出杂味 |
| 蜜处理豆（甜感明显） | 按中烘标准做，甜的就出来了 | 中烘参数，注意均匀萃取 |
| 厌氧 / 重发酵豆 | 萃取均匀点、别冲太狠；味道怪就磨粗水凉 | 注意布粉均匀、控制萃取，避免过萃出发酵杂味 |
| 新豆（刚烘几天） | 先放几天再喝（让气跑一跑），急着想喝就正常做、别磨太细 | 养豆 3–7 天排气后再用；新豆易萃不均，注意布粉 |
| 老豆（放了一两月+） | 粉磨细一点、水烫一点，味道淡了就多放粉 | 调细研磨、升温、提高粉量，对抗风味流失与萃取不足 |

## 四、新鲜度与保存

### 新手版
- 越新鲜越好喝，但刚烘好的头几天"气太多"反而怪，放 3–7 天再喝最稳。
- 买回来密封好、放阴凉不晒太阳的地方；别放冰箱（容易受潮串味），除非要存很久。
- 开封后尽快喝完（2–4 周内最佳）。

### 资深版
- **养豆期**：浅烘 5–10 天、深烘 3–5 天排气后风味稳定。
- **赏味期**：烘焙后 2–4 周为风味峰值，意式可略长。
- **保存**：单向排气阀袋 / 密封罐，避光阴凉（15–25℃）；长期（>1 月）可分装冷冻，用前整包回温至室温再开封，防结露。
- 湿度是敌人：受潮豆易出杂味、萃取不均。

## 五、新手速记口诀卡（主动展示给用户）

> **深烘磨粗温要低，浅烘磨细温要高；**
> **新豆要醒别急着，老豆磨细升点温。**

## 六、风味偏好 → 选豆方向（按口味选豆）

用户说"我喜欢巧克力味的""我喜欢果香偏酸的"，按下列映射给方向。**先让用户想清楚"想喝起来像什么"，再选豆，比盲买省一半试错。** 左栏新手白话，右栏资深参数。

| 想要的味道 | 新手白话（买什么豆） | 资深参数（产区/处理法/烘度） |
|------|------|------|
| 巧克力、坚果、焦糖、醇厚 | 选中深烘、味道"厚实偏苦香"的，别买太酸的 | 中深烘拼配；巴西/哥伦比亚 水洗；追求甜感与醇厚 |
| 花香、果香、茶感、清爽 | 选浅烘、袋上写"莓果/柑橘/茉莉"的 | 浅烘单品；埃塞俄比亚/肯尼亚；日晒或蜜处理突出香气 |
| 酸甜、果汁感、爆炸风味 | 选浅烘、袋上写"热带水果/发酵感"的 | 浅烘；哥伦比亚蜜处理、埃塞俄比亚日晒；厌氧处理更浓烈 |
| 平衡、不挑嘴、好入口 | 选中烘、写着"均衡/红糖"的 | 中烘；云南/巴西/哥伦比亚；容错最高，新手首选 |

**引导话术**：
- 新手："先想想你平时爱喝哪种——像巧克力那种厚实的，还是像水果那种清爽带酸的？照上表选豆就行。"
- 资深：可进一步按海拔/品种微调——高海拔→更明亮酸质；瑰夏→花香茶感；波旁→甜净；拼配→稳定均衡。

> 例：用户"我就想喝点果香、酸酸甜甜的"→ 推荐浅烘单品（埃塞俄比亚/肯尼亚，日晒或蜜处理），手冲最能体现；别用中深烘拼配，会压住果酸。

## 七、使用说明

- 新手讲豆子一律用大白话："炒得深/浅""放久没""果味重不重"，**不出现**烘焙度/处理法/养豆期/赏味期等词（需替换见 [glossary.md](glossary.md)）。
- 资深可同时给品种、处理法、海拔等维度，并说明它们如何共同影响萃取与风味。
- 换豆子本身就是一个变量——先照旧参数做一杯确认是豆子问题再调（调整原则见 SKILL.md「铁律」）。
