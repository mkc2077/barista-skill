# 特调咖啡（Craft / signature coffee —— 独立大类）

> 特调在本技能里是**独立的一大类**，不是"奶咖的延伸"。它有自己完整的基底萃取规范、茶底规范、自制辅料规范、采购辅料清单与 SOP（标准操作步骤）。
> **铁律：本文件给"框架 + 已核实的方法名与门店索引"，任何具体克数/萃取参数/门店当下配方必须再次联网核实该门店/博主当前公开版本后才写入回复，并标注来源链接 + 获取日期。** 特调迭代极快、季节限定多；门店菜单常换。永远以门店当下菜单/官方账号为准。

## 一、特调是什么 / 怎么分类

特调 = **咖啡基底 + 辅料 + 风味/呈现**，用自家/门店独有的搭配形成招牌。按基底分两大族：

| 基底族 | 咖啡液来源 | 典型代表 |
|--------|-----------|----------|
| **浓缩族**（Espresso base） | 意式浓缩 / ristretto | 可可拿铁、SOE Dirty、寺右限定 Dirty |
| **浸泡/滴滤族**（Infusion base） | 手冲 / 冷萃 / 冰冲 | 茉莉冷萃、争气芒芒、ALOHA（清饮气泡类） |

> 关键：同一款特调配同一款咖啡液**萃取方案不变**才能复现。基底配方在网上门店特调里往往只写"咖啡液"，**必须自己补全萃取方案**。

## 二、咖啡基底萃取方案规范（写配方时必须标明）

每款特调写配方时，**第一行就标明基底萃取方案**（四选一/多种），并给完整参数。这是 SOP 能复现的前提。

### A. 中深烘浓缩（Classic / milk-base espresso）
- **场景**：奶基特调、可可/坚果/巧克力的浓体特调。需要压苦、需要厚 crema 当奶泡与分层支撑。
- **豆**：中深烘拼配（哥伦比亚/巴西/危地马拉等配比），甜厚压苦。
- **萃取方案**（通用起步，联网核实门店豆再定）：粉 18g / 出液 36g（1:2）/ 92–94℃ / 9 bar / 25–30s。
- **变体**：可拉长到 1:2.5（lungo 段）让基底更淡更顺；可收到 1:2 不到一秒即停（只取中段），具体看辅料甜度。

### B. 中浅烘 SOE ristretto（Single Origin Espresso，短萃）
- **场景**：突出"这支豆本身风味"的特调——果香 SOE 澳白、Dirty 浅烘、清饮特调需要明亮酸香时。
- **豆**：中浅烘单一产地（埃塞水洗/日晒、肯尼亚、瑰夏等），喜欢花果酸香。
- **萃取方案**：粉 18–20g / 出液 18–27g（1:1–1:1.5，ristretto 段）/ 93–95℃ / 9 bar / 20–25s。**只取前中段，避开尾段苦**。
- **变体**：要更干净时只取到 1:1（18g/18g），把 crema 一起注入；要更厚时拉到 1:1.5。
- **注意**：SOE 浅烘 crema 较薄（联网已核实，前街实测：同参数浅烘油脂明显少于中深烘），分层能力弱，需调整呈现策略。

### C. 手冲基底（Pour-over base）
- **场景**：清饮/茶咖特调、想要干净果酸当底；ecekstropical/花香类特调。
- **萃取方案（沿用通用金杯起步）**：粉 15–20g / 总水 225–300g（1:15–1:16）/ 90–92℃ / 2:00–2:45。滤杯按风味取向（V60 锥形=明亮酸香；Kalita 波浪=圆厚甜感；详见 champion-brewing 第二节）。
- **变体**：冰手冲——粉水比压到 1:8–1:10，注成"浓液"再遇冰稀释（冰重≈水的等量）；详见 recipes-baseline 冰手冲段。

### D. 冷萃基底（Cold brew base）
- **场景**：低酸、顺滑、果香/茶香特调；茉莉冷萃、ALOHA 这类。
- **萃取方案**：粉 1:8–1:12（冷水浸泡）/ 冷藏 12–24h / 粗研磨。完成后过滤、可二次稀释到 1:15 饮用浓度。
- **变体**：冰滴（Dutch）——粉 1:10–1:15，冰水滴 1 滴/秒，6–12h；body 更柔但更酒感。

> **写特调配方时这四项里选了哪一项，必须照此格式写全**：豆种+烘焙度 / 粉量+出液+比例 / 水温+压力 / 时间。

## 三、茶底方案（含茶底的特调必须写明）

很多门店特调用**茶**做第二基底（茉莉拿铁、乌龙+咖啡、奶茶风特调），茶底与咖啡分两套 SOP，必须**单独标明**。

| 茶类 | 萃取方案（通用起步，需联网核实门店当款） | 用途 |
|------|------------------------------------------|------|
| 绿茶/茉莉（茶汤） | 茶水比 1:50–1:80 / 80–85℃ / 1.5–2.5min | 茉莉冷萃、清饮特调 |
| 乌龙（铁观音/大红袍/单丛） | 茶水比 1:40–1:60 / 90–95℃ / 1.5–3min（多次冲泡取第二泡为主） | 奶茶特调、风味奶茶 |
| 红茶（锡兰/阿萨姆/伯爵） | 茶水比 1:40–1:60 / 92–95℃ / 2–4min | 经典奶茶、伯爵咖啡 |
| 冷泡茶 | 茶:水 1:100 / 冷藏 4–8h | 低涩茶底、果茶特调 |
| 茶浓缩液（Tea concentrate） | 茶水比 1:20 短时高浓冲 → 做基底糖浆用 | 高效率入特调 |

> 茶底与咖啡液**分别萃取、分别标克数**，最后按 SOP 拼装。两者都温时注意温度匹配，避免咖啡液热倒进冷茶汤破坏风味。

## 四、自制糖浆 / 辅料方案（必须给完整做法 SOP）

门店特调里**辅料多半是自制**而非市售——糖浆、果泥、咖啡柚子酱这类。复刻时必须给**自制 SOP**，不写"适量糖浆"。

### 通用自制糖浆 SOP
- **比例**：糖:水 = 1:1（重糖浆）或 1:2（轻糖浆）。
- **做法**：冷水入锅 → 加糖搅匀 → 小火加热至糖全溶（不要沸腾）→ 离火 → 冷却 → 装消毒瓶 → 冷藏≤2 周。
- **风味糖浆变体**：
  - **香草糖浆**：基础糖浆里加 1 根剖开的香草荚（或 1 tsp 香草精），微火煮 10min，泡 1h 再过滤。
  - **焦糖糖浆**：糖干煮至琥珀色（约 170℃）→ 离火缓慢加热水（防飞溅）→ 搅匀冷却。
  - **生姜糖浆**：50ml 水加 30g 姜片煮 10min → 加 50g 糖溶 → 过滤。
  - **肉桂/八角糖浆**：基础糖浆加 1 根肉桂或 2 颗八角，浸泡 2h。

### 果泥 / 果酱 SOP（如芒果泥、莓果酱）
- **芒果泥**：新鲜芒果 200g + 糖 20g + 柠檬汁 5ml → 打成泥 → 微火煮稠 → 冷藏。
- **莓果酱**：莓果 200g + 糖 40g + 柠檬汁 10ml → 小火煮 10–15min 至果胶释出 → 冷藏。
- **咖啡果皮糖浆（ cascara syrup）**：咖啡果皮 50g + 水 250ml + 糖 100g → 小火煮 10min → 泡 30min → 过滤。

### 用油 / 酒类辅料
- **橙皮油**：橙皮只用有色部分，进配方前用喷火器轻炙去苦。
- **利口酒**（咖啡利口酒、橙酒、朗姆）：用量 5–15ml/杯，盛后加；含酒特调需注明"含酒精、限成人、孕妇忌饮"。

## 五、采购辅料清单（需自己买的果汁、椰子水、气泡水等）

这些**不做自制**，但采购要点与品牌取向需注明：

| 辅料 | 采购要点（通用） | 注意 |
|------|------------------|------|
| 椰子水 | 100% 椰子水，无添加糖 | 不同品牌甜度差大，配方需标品牌或按甜度再校 |
| 气泡水 | 高碳酸、钠含量低（如 San Pellegrino、Gerolsteiner） | 进杯避免久置失气 |
| 鲜榨果汁 | 现榨为佳；橙汁/柠檬汁/百香果汁/西柚汁常见 | 标明"现榨"或"100% NFC" |
| 牛奶/植物奶 | 全脂牛奶奶泡厚；燕麦奶需选能打泡的 barista 版 | 不同品牌油脂/甜度差 |
| 枫糖 / 蜂蜜 | 风味糖浆替代 | 甜度比白糖低，比例需校 |
| 可可粉 / 抹茶粉 | 食品级、无糖原粉 | 抹茶粉需先过筛 |

> 给用户建议时，**自制的归第四节、买的归第五节**，别混。每款特调里两者都要列清楚。

## 六、特调 SOP 框架（每款特调必须给完整操作步骤）

特调 SOP ≠ 仅配方。完整的 SOP 结构如下，**给用户/写门店复刻时必须填满**：

```markdown
【特调 SOP 模板】
特调名：{name}（来自 {门店/博主}，{城市}，截至 {YYYY-MM-DD} 菜单）

1. 咖啡基底（必填）
   - 类型：{A 中深烘浓缩 / B SOE ristretto / C 手冲 / D 冷萃}
   - 豆种：{产区/烘焙度/拼配或SOE}
   - 萃取参数：粉 {g} / 出液或总水 {g}（{ratio}） / 水温 {℃} / 压力 {bar} / 时间 {s}
   - 备注：{只取中段 / 是否带 crema / 是否预先稀释}

2. 茶底（仅含茶时必填，否则写"无"）
   - 茶类 + 茶水比 + 水温 + 时间
   - 是否冷泡 / 是否取第二泡

3. 自制辅料（按使用顺序逐项写做法 SOP）
   - 辅料1：{名称} 克数 {g} — 做法 SOP（见第四节的哪条变体）
   - 辅料2：……

4. 采购辅料（逐项写，注明品牌/规格取向）
   - 椰子水 {g} / 气泡水 {g} / 鲜榨橙汁 {g} / ……

5. 杯具与冰
   - 杯子：{360ml 玻璃杯 / 300ml 高球 / 250ml 陶瓷拿铁杯}
   - 冰：{无冰/普通冰 100g/大方冰 80g/碎冰}

6. 拼装 SOP（按顺序、带口诀）
   ①  {先入杯什么}  → 例：糖浆 {g} 入杯底
   ②  {第二入什么}  → 例：冰块入杯至八分
   ③  {第三入什么}  → 例：现榨果汁沿冰块缓注（沿杯壁流，不到底）
   ④  {第四入什么}  → 例：咖啡液顶部缓注（保持分层）
   ⑤  {收尾}        → 例：橙皮油喷炙 + 搅棒斜搭
   口诀："{一句好记}-{一句好记}{句口}"

7. 呈现与饮用提示
   - 是否搅拌/分几口喝/分层顺序
   - 饮用窗口：完成 X 分钟内风味最佳
   - 含酒精注明

8. 来源
   - [{title}]({url})，获取于 {YYYY-MM-DD}
   - 复刻时"截至此为菜单"，门店常换以官方为准
```

## 八、意式特调通用要点（先看这个再细化）

- **豆子**：奶基特调常用中深烘拼配（甜厚、压苦）；突出豆子本味（SOE 澳白/特调）用浅烘 SOE。详见 [recipes-baseline.md](recipes-baseline.md) 第九节。
- **辅料顺序**：糖浆/果泥先入杯，浓缩后注，再注奶/奶泡/冰——避免油脂被破坏。
- **冰饮**：冰块先称量，热萃遇冰会瞬间稀释，特调尤其要控冰量。
- **风味平衡**：辅料有甜度时，浓缩/基底要相应偏浓（如用 ristretto 段），避免太腻。
- **呈现**：分层靠密度差（糖浆沉底、奶中、泡沫顶），Dirty 是典型"不动"分层（奶在下、浓缩缓注入上）。

## 九、给建议的输出格式（含完整 SOP）

按第六节 SOP 模板填满所有项后输出，结尾固定写：
> "门店特调是参考，家里复刻按手头器具和口味微调。调整原则见技能铁律。"

## 十、常见误区

- ❌ **照抄门店配方不问豆**：门店特调的豆子是定制的，家用要换成你手头相近烘焙度。
- ❌ **不标萃取方案就给配方**：没写"中深烘浓缩 18g/36g"或"SOE ristretto 18g/22g"，特调必然不可复现。
- ❌ **不区分自制与采购辅料**："加糖浆"等于什么糖浆多少克谁做的，必须明确。
- ❌ **只有配方没有步骤**：特调靠拼装顺序与温度，省略 SOP 必失败。
- ❌ **用太浓的辅料盖过咖啡**：特调讲平衡，咖啡要还在，不是糖水。
- ❌ **凭名字编造克数/比例**：所有数字必须来自联网核实并标来源。
- ❌ **不分季节限定与常驻**：很多招牌是季节款，给时注明"截至 XX 为菜单"。
- ✅ **先抄框架（基底+辅料类目+茶底），再按自己舌头调比例与 SOP**。
