# 冠军冲煮方案索引（Champion brewing recipes）

> **铁律：本文件只提供"已联网核实的名家姓名 + 比赛头衔 + 核心方法名"，任何具体数字（粉量/水温/比例/压力/时间）必须再次联网核实该名家的当前公开方案后才写入回复。**
> 名家会随年份迭代手法；比赛方案与家用方案不同。给用户前务必补一条**来源链接 + 获取日期**。

## 一、SCA 冲煮与金杯标准（先行校准）

- **金杯（Golden Cup）**：萃取率 18–22%、TDS 1.15–1.35%、粉水比 1:18–1:19（手冲常用区间更宽到 1:14–1:17）。详见 [parameters-guide.md](parameters-guide.md) 第一节。
- **WBrC（World Brewers Cup，世界冲煮大赛）**：两个 service——Open Service（选手自带豆 + 自选方案）与 Compulsory Service（组委会统一豆 + 闭门冲煮）。评分含风味、技术与presentation。
- **养豆**：浅烘建议养豆 7–15 天，中深烘 5–10 天（再次提醒：此为通用区间，具体豆需联网/问烘焙商）。

## 二、名家冲煮方案索引（联网检索起点）

> 下表为"检索起点"；给建议时务必先按"机型 / 豆子 / 烘焙度"再查该名家的当前方案。

### 1. 粕谷哲 Tetsu Kasuya —— 4:6 法（四六法）
- **头衔**：2016 WBrC 世界咖啡冲煮大赛冠军（亚洲首位手冲世界冠军）。
- **核心**：把总注水量拆成 **前 40% 控酸甜 + 后 60% 控浓淡**。
- **标准基线（网上流传最广版本，使用前请再核实）**：粉 20g / 总水 300ml / 粉水比 1:15；前 40% = 120ml（如 60ml × 2），后 60% = 180ml（如 60ml × 3）；研磨偏粗。
- **风味调整逻辑（已核实要点）**：
  - 太酸/尖酸 → 前 40% 分段减少（3→2 或 2→1），让前段更集中、甜感更早出现。
  - 太淡/不够厚 → 后 60% 分段更集中（段数变少、每段水量加大），让浓度堆叠更快。
  - 整体浓淡 → 先动粉水比（1:16 更清爽、1:14 更醇厚）。
  - 一次只改一件事，顺序：段落（前40/后60）→ 粉水比 → 水温/研磨。
- **可用产出**：《就這麼簡單！世界冠軍親授「4：6法」手沖奧義全解析》（方舟文化，2023）。
- **检索**：`Tetsu Kasuya 4:6 method` / `粕谷哲 4:6 法` / `Kasuya 4:6 ratio pour over`。

### 2. 王策 Chad Wang —— VWI by CHADWANG
- **头衔**：2017 WBrC 世界咖啡冲煮大赛冠军。
- **核心**：强调高粉水比、长萃取、用浸泡/稳定控温结合，追求干净度与层次。
- **检索**：`Chad Wang VWI brew recipe` / `王策 冲煮 World Brewers Cup 2017` / `VWI by CHADWANG signature`。

### 3. 杜嘉宁 Du Jianing / "豆子" —— UNiUNi
- **头衔**：2019 WBrC 世界咖啡冲煮大赛冠军（波士顿）；**中国首位世界咖啡锦标赛冠军**。
- **方法要点（联网核实）**：UNiUNi 南京；2018 世锦赛止步八强后重整 routine；她在 presentation 里让评委先品饮再给信息（先喝再讲）；技巧包含"把豆子磨两次"以提细粉/萃取、并为评委多冲一杯一起喝。
- **检索**：`Du Jianing World Brewers Cup 2019 routine` / `杜嘉宁 冲煮 routine` / `UNiUNi brew recipe`。来源起点：Barista Magazine 专访。

### 4. 吴则霖 Berg Wu —— 三温暖手冲法
- **头衔**：2016 WBC 世界咖啡师大赛冠军；兴波咖啡 Simple Kaffa 共同创办人。
- **方法名**："三温暖手冲法"——在萃取中用**三种不同温度**分段冲煮。
- **公开演示要点（网上流传版本，使用前请再核实）**：埃塞俄比亚水洗 / 16g 粉 / 240g 总水；Clever 聪明杯；闷蒸 30s；第一阶段接近沸腾的高温水，快滴完时加冷水降到约 88–93℃；第二段注到 200g；再移开下壶加冷水降到约 80–85℃；最后 40g 水浸泡 20s。希望酸更亮→降温偏高；希望甜更重→降温偏低。
- **检索**：`Berg Wu three temperature pour over` / `吴则霖 三温暖 手冲` / `Simple Kaffa brew`。

## 六、常见误区

- ❌ **直接照抄比赛方案**——比赛豆、磨豆机、水质与家用不同，需做基线对照。
- ❌ **只记名字不记版本**——同一人不同年份的方案差别很大，务必带年份查。
- ❌ **把"世界中变"误当"不变真理"**——4:6、三温暖都是可调框架，不是唯一解。
- ❌ **凭名字编造数字**——所有粉量/水温/比例必须来自联网核实。
- ✅ **先用通用起步参数做基线，再只改一个变量**叠加名家手法。
