# 咖啡研磨度校准指南

> 研磨是咖啡萃取的第一个关键变量。本文件系统讲解研磨科学原理、不同磨豆机的校准方法及常见问题解决。
>
> **新手转述**："磨豆机就像调吉他——用之前要先校准好，不然出来的味道怎么调都不对。"

## 零、研磨度的科学原理

### 0.1 为什么研磨很重要

咖啡豆约 **28%** 是水溶性的，其余为纤维素和植物结构物质。研磨决定了水与咖啡接触的**表面积**——研磨越细，表面积越大，萃取越快；研磨越粗，表面积越小，萃取越慢。

### 0.2 化合物溶出顺序

烘焙咖啡中的水溶性化合物按固定顺序溶出：

**果酸类（最先）→ 脂类 → 糖类（甜味）→ 碳水化合物（苦味，最后）**

这就是为什么萃取不足的咖啡偏酸偏咸（糖类还没溶出），过萃的咖啡偏苦偏涩（碳水化合物过度溶出）。

### 0.3 研磨粗细与萃取关系

| 研磨粗细 | 表面积 | 萃取速度 | 风险 | 适用场景 |
|---------|--------|---------|------|---------|
| 极细（面粉状） | 最大 | 极快 | 过萃、通道效应 | 土耳其咖啡 |
| 细（细盐状） | 大 | 快 | 需精确控时 | 意式浓缩 |
| 中细（白砂糖） | 中等 | 适中 | 较安全 | 手冲、冰手冲 |
| 中等（粗砂糖） | 中偏小 | 稍慢 | 可能寡淡 | 聪明杯、爱乐压 |
| 粗（海盐） | 小 | 慢 | 欠萃风险 | 法压、冷萃 |

### 0.4 粒径分布与均匀度（关键原理）

咖啡萃取率取的是所有颗粒的**平均值**。不同大小的颗粒萃取程度不同：

- **颗粒均匀度高**（如 Mahlkönig EK43）：萃取率可达 21%–26% 而不出现不良风味
- **颗粒大小范围广**（不均匀）：
  - 大颗粒（boulders）萃取不足 → 贡献酸涩、植物性味道
  - 最小颗粒（fines）过萃 → 释放苦涩和干涩物质
  - 结果：**同时出现酸涩和苦味**——这就是低端磨豆机"怎么调都不好喝"的根因

> 这是投资高品质磨豆机的核心理由：均匀度越高，可萃取窗口越宽，风味越清晰。

### 0.5 细粉问题

细粉（fines）是粒径极小的颗粒，萃取极快，容易产生不良风味。低品质磨豆机会产生过多细粉。研磨并非越细萃取越高——超过某个细度后，细粉结块导致微通道效应（micro-channeling），反而降低萃取率。

## 一、通用校准原则：Dose → Yield → Time

基于行业标准的系统校准方法：

| 步骤 | 操作 | 原则 |
|------|------|------|
| 1. 固定粉量 Dose | 确定每次使用的咖啡粉量（如意式 18g，手冲 15g） | 粉量第一，不可妥协 |
| 2. 固定液重 Yield | 确定目标萃取液重量（如意式 36g = 1:2） | 液重第二，目标不可妥协 |
| 3. 调整时间 Time | 通过改变研磨度调整萃取时间至目标范围 | 时间灵活：调粗=加速，调细=减速 |

**关键原则**：
- 粉量第一，液重第二，时间第三
- 在不触及边际递减点的前提下，尽可能调细
- 改变研磨度后必须 purge（冲洗/吹出）旧粉
- 持续萃取更多，直到不再改善为止，然后退回一步——即到达最优萃取点

## 二、手摇磨豆机校准（以 Comandante C40 为例）

C40 拥有 35 格（MK4 为 40 格）精细调节系统和高氮马氏体不锈钢刀盘。

### 2.1 研磨度推荐表

| 冲煮方式 | 建议刻度 | 颗粒参考 |
|---------|---------|---------|
| 土耳其咖啡 | 5–10 格 | 极细粉（面粉状） |
| 意式浓缩 | 10–15 格 | 细盐状 |
| 手冲咖啡 | 20–25 格 | 白砂糖颗粒 |
| 法压 / 冷萃 | 30–35 格 | 粗砂状 |

### 2.2 归零校准步骤

1. 旋转调节环至刀盘完全闭合（听到金属摩擦声），此为"零点"
2. 逆时针旋转 1 圈约对应 20 格，适用于 V60 手冲
3. 2 圈（约 24 格）适合法压壶
4. 记录你的零点位置，后续以此为基准

### 2.3 其他手摇磨参考刻度

| 磨豆机 | 意式起步 | 手冲起步 |
|--------|---------|---------|
| 1Zpresso JX-Pro | 1.0–1.2 圈 | 2.0–2.4 圈 |
| 1Zpresso K-Plus | 1.0–1.5 圈 | 2.2–2.6 圈 |
| Timemore C2/C3 | 8–10 格 | 14–18 格 |
| Timemore Chestnut X | 10–12 格 | 14–18 格 |

## 三、意式电动磨豆机校准（以 Mahlkönig EK43 为例）

EK43 采用 98mm 钢合金平刀盘，立式设计，是精品咖啡行业标杆。

### 3.1 意式刻度校准步骤

1. 将刻度调整到 **1** 的位置
2. 用内六角扳手拧松刻度旋钮上的螺丝（共 2 个）
3. 磨豆机开启状态下，用一字螺丝刀顺时针慢慢调小刀盘间隙，直到听见刀盘轻微摩擦声
4. 再逆时针调粗一点点，找到零界点（无摩擦声）
5. 关闭磨豆机，用内六角锁死刻度旋钮螺丝

### 3.2 校准后使用范围

- SOE 意式研磨：**1.3–1.8**（根据烘焙度和粉量调整）
- 单品手冲：**7–8.5**（根据烘焙度和冲泡手法调整）

### 3.3 高级校准方法（Matt Perger 方法）

1. 用白板笔在刀盘周围做标记
2. 故意调整至刀盘轻微摩擦
3. 拆开观察标记上的不均匀擦除痕迹
4. 在磨损痕迹的对角处垫纸片，装回再测试
5. 反复调整直到刀盘摩擦后标记均匀擦除

### 3.4 Eureka 等家用电动磨豆机

- 采用锥形或平刀盘设计，校准方法类似：找零点（刀盘接触）→ 以此为基准设定研磨度
- 意式研磨通常在零点以上数格范围内调整
- 注重低残留设计，减少换豆时的交叉污染
- 惠家 ZD-17/ZD-18N：数字档位，意式约 2.5–3.5 档
- Baratza Sette 270：30 档，意式约 5–10 档

## 五、校准实操流程

### 5.1 意式浓缩校准流程

**步骤 1：设定基准参数**
- 粉量：18g
- 目标液重：36g（1:2 比例）
- 目标时间：25–32 秒

**步骤 2：品尝与诊断**
- 太酸太薄 → 萃取不足 → 调细研磨
- 太苦太涩 → 过萃 → 调粗研磨
- 强度不够 → 增加粉量或减少液重

**步骤 3：微调**
- 每次只调一个参数
- 调整研磨度后充分 purge
- 记录每次调整的结果

### 5.2 手冲校准流程

**步骤 1：设定基准**
- 粉量：15g
- 总水量：225g（1:15）
- 水温：92°C
- 研磨度：中等（C40 约 22 格）

**步骤 2：诊断与调整**
- 风味寡淡/酸涩 → 调细研磨或提高水温
- 风味苦涩 → 调粗研磨或降低水温
- 浓度太低 → 降低粉水比

## 六、常见问题与解决方案

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 萃取液流速过快 | 研磨太粗/粉量太少/布粉不均 | 调细研磨/增加粉量/改善布粉 |
| 萃取液流速过慢 | 研磨太细/粉量太多 | 调粗研磨/减少粉量 |
| 咖啡太酸 | 萃取不足 | 调细/提高水温/延长时间 |
| 咖啡太苦 | 过度萃取 | 调粗/降低水温/缩短时间 |
| 风味不清晰 | 萃取不均匀 | 改善研磨均匀度/改善布粉/使用更好的粉碗 |
| 研磨刻度漂移 | 长期使用导致机械松动 | 定期重新校准零点 |
| 清洁后研磨不均 | 刀盘螺丝未锁紧导致倾斜 | 完全锁紧螺丝并重新校准 |
| 同时出现酸涩和苦味 | 粒径分布不均（细粉+大颗粒并存） | 使用更均匀的磨豆机/筛除细粉 |
| 换豆后味道突变 | 新旧豆残留混合 | 充分 purge 旧粉后再调参 |
| 静电大/粉飞溅 | 干燥天气 | 用 RDT（喷雾瓶轻喷 1–2 下水再磨） |
| 粉结块/细粉过多 | 豆子含水率高、刀盘残留 | 养几天再用/清理刀盘残留/检查刻度环松动 |

## 七、提高萃取均匀度的方法

1. 使用高品质粉碗（如 VST 精密粉碗）
2. 使用完全平整的压粉器，确保压粉水平
3. 压粉前充分布粉，粉面应像高尔夫果岭般平整
4. 插入手柄时不要敲击（会导致通道效应）
5. 使用不会产生过多大颗粒的磨豆机（平刀通常优于锥刀）
6. 避免使用溶解度差异过大的拼配豆
