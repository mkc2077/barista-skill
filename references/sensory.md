# 感官与风味调整映射

本文把"喝起来怎么样"翻译成"下一步调什么"。左栏为新手大白话版，右栏为资深参数版。
回复时按用户经验档位选对应一栏，不要混用。

## 零、如何品咖啡（主动教用户描述味道）

不要等用户问才讲"怎么尝"。给出方案或调整后，主动引导用户描述喝到的味道。

### 新手版（大白话三步尝味法）
1. **先闻**：香不香？像坚果、巧克力，还是水果？
2. **喝一口，让咖啡在嘴里转一下**：是酸、是苦、还是甜？
3. **吞下去后**：嘴里还留不留香味（这叫"回甘"）？
> 用大白话描述即可，例如"有点酸、不太苦、喝完嘴里甜甜的"，不需要专业词。

### 资深版（专业品鉴六维度）
从以下维度描述，可用啜吸(slurp)让咖啡雾化以捕捉干香/湿香：
- **Aroma 香气**：干香、湿香
- **Acidity 酸质**：明亮/圆润/尖锐
- **Sweetness 甜感**：明显/隐蔽
- **Body 醇厚/口感**：轻盈/厚重/顺滑
- **Aftertaste 余韵**：长短、干净与否
- **Balance 平衡感**：各维度是否协调

## 一、味道问题 → 调整动作

| 喝起来的感觉 | 新手白话版（调什么） | 资深参数版（调什么） |
|------|------|------|
| 太苦、焦苦、中药味 | 粉磨**粗**一点；水少泡一会儿（缩短时间）。口诀：**苦调粗** | 调粗研磨 / 降低水温 1–2℃ / 缩短萃取时间 / 降低粉量比；排查过萃 |
| 太酸、尖酸刺舌、青涩 | 粉磨**细**一点；多泡一会儿（延长时间）。口诀：**酸调细** | 调细研磨 / 升高水温 / 延长萃取时间 / 提高粉量比；排查欠萃 |
| 淡如水、没味道、空 | 粉磨细一点，或**少放水多放粉** | 提高粉量、调细研磨、降低粉水比（如手冲 1:15→1:14） |
| 太浓、厚重发闷 | 粉磨粗一点，或**多放水少放粉** | 降低粉量、调粗研磨、提高粉水比 |
| 水流出太快、咖啡很稀 | 粉磨**细**一点 | 调细研磨、提高粉量、检查布粉均匀度 |
| 水半天流不出、像堵住 | 粉磨**粗**一点 | 调粗研磨、减少粉量、检查填压力度 |
| 涩口、干巴巴刮舌头 | 粉磨粗一点，水别太烫 | 调粗研磨、降水温、排查通道效应与过萃 |
| 闻着香喝着没味 | 多放点粉、粉磨细一点 | 提高粉量、调细研磨、确认水温与新鲜度 |

## 三、风味描述词典（资深用，新手需转述）

### 3.1 基础风味词

| 风味词 | 含义 | 通常指向 |
|------|------|------|
| 明亮 / 尖酸 | 酸质突出、活泼 | 可能欠萃或浅烘豆 |
| 醇厚 / Body 重 | 口感厚重顺滑 | 深烘、细研磨、高粉量 |
| 干净 / 清晰 | 风味层次分明无杂味 | 均匀萃取、好水质 |
| 尖涩 / 收敛 | 口干刮舌 | 过萃、通道效应、水温偏高 |
| 空洞 / 寡淡 | 无风味 | 欠萃、粉水比过高 |

## 四、咖啡风味轮的构成原理与使用方法

咖啡风味轮由 SCAA 联合世界咖啡研究组织（WCR）于 2016 年完成首次重大更新，基于《世界咖啡感官研究大辞典》（WCR Sensory Lexicon），该辞典识别出 110 种风味、香气和质地属性。

### 4.1 风味轮的分层结构

| 层级 | 内容 | 说明 |
|------|------|------|
| 最内圈（中心） | 五种基本味觉 | 甜、酸、咸、苦、鲜（注：涩是触感非味觉，不列入） |
| 第一环 | 九大风味类别 | 花香、水果、酸感/发酵、绿色/蔬菜、烘烤、香料、坚果/可可、甜感、其他 |
| 第二环 | 子类 | 每个大类延伸出更具体的子类描述 |
| 最外圈 | 具体描述词 | 约 99 种精细风味描述词 |

### 4.2 "缝隙距离"原理（关键使用方法）

- **相邻且紧密相连**：两个风味项之间"缝隙"越窄，关联度越高、越相似
- **存在间隙/距离越大**：缝隙越宽，关联度越弱、差异越大
- **从内向外**：越往外圈，风味间的相似性和关联度越来越弱

### 4.3 正确使用方法

1. **从中心向外检索**：先确定基本味觉 → 再找所属大类 → 逐步定位到具体描述词
2. **结合感官词典使用**：SCAA 建议将风味轮与 WCR 感官词典一同使用，词典中每个风味项都有明确定义和校准参照物
3. **作为行业通用语言**：风味轮的核心意义在于提供统一的共同语言，让不同地区、不同岗位的从业者能无障碍沟通咖啡感官信息

## 五、系统化感官训练方案

### 5.1 基础味觉识别训练

根据 SCA 感官基础课程标准，五种基本味觉的标准训练溶液配制：

| 基本味觉 | 溶液配制（1L 水中溶解） |
|----------|----------------------|
| 甜 | 24g 蔗糖 |
| 酸 | 1.2g 柠檬酸 |
| 咸 | 4g 氯化钠（食盐） |
| 苦 | 0.54g 咖啡因 |
| 鲜 | 2g 味精（谷氨酸钠） |

**进阶强度训练**：将每种味觉配置成 4 个强度等级（1 最弱，4 最强），进行分层品尝和盲测：
1. 独立品尝每种强度的酸甜咸
2. 同类、同强度混搭；同类、不同强度混搭
3. 不同类、不同强度混合，体会味道间的掩盖与增强效应
4. 盲喝测试——辨别混合溶液的具体物质及强度

**咖啡酸质专项（参照 CQI Q-Grader 标准）**：
- 柠檬酸：柑橘类酸质（柠檬、柳橙）
- 苹果酸：青苹果型酸，清爽酸甜
- 酒石酸：葡萄型酸
- 醋酸：如醋般刺激、不愉悦的酸
- 绿原酸：烘焙中分解为奎宁酸与咖啡酸

### 5.2 嗅觉记忆建立方法

**关键认知**：常说的"苹果味"并非味觉，而是**鼻后嗅觉**（retronasal olfaction）从口腔闻到的气味。可通过"棒棒糖实验"区分——捏住鼻子吃水果味棒棒糖只能尝到甜味，松开鼻子瞬间风味明了。

**Le Nez du Café（咖啡鼻子）36 味闻香瓶**：由法国 Jean Lenoir 发明，SCA 认证课程将其作为 Q-Grader 必考项。按烘焙化学变化分四大群组：

| 群组 | 对应烘焙阶段 | 香气特征 | 代表香气 |
|------|------------|----------|---------|
| 酶催化组 | 浅烘焙 | 花香、果香、草本香 | 玫瑰花香、咖啡花、黄柠檬、杏桃、苹果 |
| 焦糖化组 | 中烘焙 | 坚果、焦糖、可可香 | 蜂蜜、焦糖、黑巧克力、烤杏仁、烤榛果 |
| 干馏组 | 深烘焙 | 树酯、香料、碳烧香 | 丁香、胡椒、烟草丝、烟熏味 |
| 瑕疵组 | — | 不愉悦的瑕疵风味 | 土味、马铃薯、青豆、橡胶味 |

**日常嗅觉记忆建立**：利用常见食材建立嗅觉记忆库——水果类（柠檬、莓果、桃子）、花香类（玫瑰、茉莉）、坚果类（杏仁、榛果）、香料类（肉桂、丁香、香草）。核心原则：大量尝试、记住生活中随处可见的物质气息。

### 5.3 对比品鉴练习

| 训练类型 | 内容 | 难度递进 |
|---------|------|---------|
| 三角杯测 Triangulation | 三杯中两杯相同、一杯不同，需找出不同的那一杯 | 不同产区→不同处理法→不同烘焙度→同农场不同批次 |
| 产区比较杯测 | 对比不同产区（非洲/中南美/亚洲）的风味差异 | 先大区对比，后同区不同国家对比 |
| 瑕疵杯测 | 训练识别不好的风味（土味、霉味、发酵过度等） | 从明显瑕疵到细微瑕疵 |
| 烘焙度对比 | 同一豆子不同烘焙度对比品鉴 | 理解烘焙度对风味的影响 |

### 5.4 个人化咖啡嗅觉与味觉记忆库搭建

1. **味觉基础库**：配置酸甜咸苦鲜标准溶液，4 级强度，反复品尝记忆
2. **鼻后嗅觉库**：通过棒棒糖/新鲜水果盲测，建立鼻后嗅觉与味觉的联动记忆
3. **干香湿香库**：闻香瓶 36 味/T100 系统训练，辅以日常食材
4. **咖啡风味映射库**：用杯测方式品鉴不同产区/处理法/烘焙度的咖啡，记录并对照风味轮定位
5. **温度变化库**：同一杯咖啡在高/中/低温下的风味变化记录
6. **感官卫生维护**：戒烟、戒酒、戒辣等刺激性食物，保持感官灵敏度

## 六、使用说明

- 新手每次给建议都要带一句口诀，并说明"把手上的研磨调钮往粗/细转一点"。
- 资深可同时给多个可调变量，并说明变量间的权衡（如调粗研磨会降低醇厚感）。
- 调整原则（一次只改一个变量等）见 SKILL.md「铁律」。
- 风味轮与感官训练内容为进阶/资深用户准备；新手只需用三步尝味法+口诀。
- 系统化感官训练（味觉溶液、闻香瓶、三角杯测）详见第五节，学习资源见 [learning-resources.md](learning-resources.md)。
- 杯测流程见 [cupping.md](cupping.md)，参数调整原理见 [parameters-guide.md](parameters-guide.md)。
