# 稳妥起步参数（新手可直接照做）

以下为"第一次做也不容易翻车"的起步方案，覆盖十七种常见做法。资深用户可据此微调。
所有方案都假设使用新鲜咖啡豆（烘焙后 2–4 周内最佳）与可称重量的电子秤。

**选什么豆子、豆子怎么影响萃取（深烘/浅烘/处理法/新鲜度），见 `references/beans.md`。**

**通用口诀（记牢）**：
> 苦调粗，酸调细；淡了粉多水少，浓了粉少水多；水流快调细，水流慢调粗。
> （完整口诀规则与新手使用指南见 SKILL.md「新手模式硬性约束」）

---

## 一、意式浓缩（Espresso）

适用：半自动咖啡机 + 磨豆机（能调粗细）

- 粉量：18 克（双份）
- 出液：约 36 克（比例约 1:2）
- 水温：92–94℃
- 时间：25–30 秒
- 研磨：中等偏细，以 25–30 秒出完 36 克为准
- 步骤：
  1. 称 18 克粉，倒进手柄，铺平。
  2. 压平（轻压即可，不用大力）。
  3. 开机，同时计时。
  4. 到 36 克左右停（约 27 秒）。
- 太苦 → 粉磨粗一点（**苦调粗**）；太酸 → 粉磨细一点（**酸调细**）

### 一-B、意式变体：单份 / Ristretto / Lungo

#### 单份 Espresso（single shot）
- 粉量：7–9g（**7g 粉碗**或单份手柄）
- 出液：14–18g（粉水比约 1:2）
- 时间：20–28 秒
- 适用：早晨一杯到位、不想要太多量。注意单份粉碗的容量，别硬塞 14g（会顶手柄盖子）。

#### Ristretto（短萃 / 浓厚）
- 粉量：18g（双份粉碗）
- 出液：18–27g（粉水比约 1:1–1:1.5，**比标准 espresso 短**）
- 时间：15–22 秒
- 适用：澳白、flat white、dirty、特调底；甜感更强、更浓缩。
- 注意：粉研磨要稍**调细**（比标准 espresso 略细）来补偿更短的萃取时间；太苦/尖酸反调。

#### Lungo（长萃）
- 粉量：18g（双份粉碗）
- 出液：54–72g（粉水比约 1:3–1:4，**比标准 espresso 长**）
- 时间：35–45 秒
- 适用：美式、想直接喝一杯大份量不加奶的人。
- 注意：粉研磨要稍**调粗**（比标准 espresso 略粗）来补偿更长的萃取时间，避免过萃发苦。

#### 美式（Americano）
- **热美式**：espresso（18g 粉 / 36g 出液）+ 热水 150–200ml，比例 1:4–1:6 自由调。
- **冰美式**：espresso 36g + 冰 + 冷水 200ml 即可。
- 进阶：先倒水再倒咖啡（保留 crema）；或先倒咖啡再倒水（更均匀）——两种流派都能喝。

### 一-C、全自动 / 胶囊机用户指引

若你用的是**全自动咖啡机**（一键出浓缩，粉量与研磨由机器固定）或**胶囊机**（Nespresso 等），本文件的"研磨 / 水温微调"对你不适用——这些由机器决定。你的可调空间在：
- **豆子 / 胶囊选择**：奶咖选中深烘拼配胶囊或豆；想喝单品风味选浅烘 SOE 胶囊。
- **杯量按钮**：espresso（小杯浓）vs lungo（大杯长）；奶咖多用 espresso 基底。
- **奶比例**：拿铁多奶、澳白少奶薄泡、dirty 不搅拌。
- **口味微调**：同一胶囊在不同机器出水温度略有差异，觉得太苦可试换"intenso/低因"之外的更浅烘焙胶囊，或加奶/水稀释。

> 全自动/胶囊机重点是"选对豆 + 调奶比"，而非调研磨；需要现磨微调时再回到半自动方案。

## 二、手冲（Pour-over）

适用：手冲壶 + 滤杯（如 V60）+ 磨豆机 + 秤

- 粉量：15 克
- 水量：240 克（比例约 1:16）
- 水温：90–92℃（深烘降到 88℃，浅烘可到 93℃）
- 研磨：中等（像粗砂糖）
- 总时长：2 分 30 秒 – 3 分 30 秒
- 步骤（分段注水）：
  1. 粉倒入滤杯，先倒 30 克水"醒一下"（闷蒸）30 秒。
  2. 分 2–3 次慢慢加水到 240 克，画圈浇在粉上、别冲到滤纸边。
  3. 等水滤完即可。
- 太苦 / 焦 → 粉磨粗一点（**苦调粗**）；太酸 / 寡淡 → 粉磨细一点（**酸调细**）

### 二-B、手冲之 Kalita Wave 蛋糕杯（新手更友好）

适用：Kalita Wave 155（1–2 人份）/ 185（2–3 人份）+ Kalita 滤纸 + 手冲壶 + 磨豆机

- 粉量：15g（155）/ 20g（185）
- 水量：240g / 320g（粉水比约 1:16）
- 水温：90–92℃（深烘降到 88℃，浅烘可到 93℃）
- 研磨：中等（比 V60 略细，Kalita 流速慢所以稍细一点让总时间合理）
- 总时长：2 分 30 秒 – 3 分 30 秒
- 步骤：
  1. 滤纸先润湿（去纸味 + 贴壁），倒掉下面的水。
  2. 粉倒入滤杯，**先倒 30–40g 水闷蒸 30–45 秒**（Kalita 流速慢，闷蒸稍长）。
  3. 分 2–3 段慢注水到目标量，注水时"中心向外 1cm 画圈"即可（别冲到滤纸边缘）。
  4. 让水自然滴完（一般 3 分钟内）。
- 优势：平底三孔流速稳定，**比 V60 宽容**，新手不容易翻车；缺点是"上限"不如 V60。
- 调整：同 V60——太苦磨粗，太酸磨细。

## 三、法压壶（French Press）

- 粉量：15 克，粗研磨（像海盐）
- 水量：250 克，水温 93℃
- 时间：浸泡 4 分钟，压下去，倒出
- 调整：太苦→粉磨更粗；太酸→粉磨细一点或延长浸泡。

## 四、爱乐压（AeroPress）

适用：爱乐压壶 + 磨豆机（最宽容、不易翻车，适合新手）

- 粉量：15–17 克
- 水量：220–250 克
- 水温：80–90℃（深烘用低一点，浅烘用高一点）
- 研磨：中细（比手冲细，比意式粗）
- 时间：浸泡约 1 分钟，再压约 20–30 秒
- 步骤：
  1. 装粉，倒水到目标量，用勺子搅两下。
  2. 闷 1 分钟。
  3. 放上滤盖，慢慢往下压（20–30 秒压完）。
- 调整：太苦→粉磨粗一点或水降温；太酸→粉磨细一点或水升温。

## 五、摩卡壶（Moka Pot）

适用：摩卡壶 + 炉灶（煤气/电磁炉均可）

- 粉量：18–20 克，中细研磨（像细砂糖偏细）
- 水：下壶装冷水，到安全阀下面一点点（别超过）
- 火：中小火
- 时间：上壶开始出咖啡约 1–2 分钟；听到"咕噜咕噜"声立刻离火
- 步骤：
  1. 下壶加冷水，粉碗装粉铺平（别压）。
  2. 拧紧上下壶。
  3. 中小火加热，听到出咖啡后转最小火收尾。
  4. 一听到"咕噜"声就关火、离炉。
- 调整：太苦/有焦味→火关小一点、粉磨粗一点、别煮到咕噜太久；有铁锈/生味→别让水烧干、粉别压太实。

## 六、冷萃（Cold Brew）

适用：任何容器 + 冰箱（夏天首选，几乎零失败）

- 粉量：50–70 克，粗研磨（像海盐）
- 水量：500–700 克，常温水或冷水
- 时间：冰箱浸泡 12–24 小时
- 步骤：
  1. 粉和水倒在一起搅匀。
  2. 盖好放冰箱。
  3. 时间到，用滤纸/滤网过滤即可喝（可加冰或水稀释）。
- 调整：太苦→粉磨更粗或时间缩短；太淡→粉多放点或时间拉长。

## 七、冰滴（Ice Drip / 冰酿）

适用：冰滴壶（或简单滴滤装置）

- 粉量：40–60 克，中粗研磨
- 冰水：400–600 克（冰 + 水混合）
- 时间：慢慢滴 4–8 小时（目标约每秒 1 滴）
- 步骤：
  1. 粉铺平在粉层。
  2. 上调阀门让冰水慢慢滴下（太快就调慢）。
  3. 接住咖啡，冷藏后喝。
- 调整：太快/太淡→把阀门调慢、粉磨细一点；太苦→粉磨粗一点、冰水比例多加水。

## 八、聪明杯（Clever Dripper）

适用：聪明杯 + 滤纸 + 磨豆机（像"泡着的手冲"，比手冲好控制）

- 粉量：15–20 克，中研磨（像粗砂糖）
- 水量：240–300 克，水温 90–93℃
- 时间：浸泡 2–3 分钟，再打开阀门滴下
- 步骤：
  1. 放滤纸，加粉，倒水到目标量。
  2. 盖好，闷 2–3 分钟。
  3. 把杯子放到杯子上，打开底部阀门，咖啡自己滴下。
- 调整：同手冲——太苦粉磨粗，太酸粉磨细。

## 九、咖啡特调、经典奶咖与冰手冲（Signature / 经典奶咖 / 创意饮 / Iced）

做特调、经典奶咖或冰手冲前，**先列配方与器材清单**，再开始。下面给常见款的安全起步。

> 新手一句话：奶咖就记——**奶越多越淡、奶泡越厚越"空气感"**；想浓选澳白/可塔朵，想顺选拿铁，想厚泡小杯选卡布。

### 经典意式奶咖比例速查表（以双份浓缩 ≈ 36–40g 为基底）

| 饮品 | 浓缩 | 热奶 | 奶泡 | 总量 | 口感定位 |
|------|------|------|------|------|----------|
| 玛奇朵 Espresso Macchiato | 1 份 | 极少（一勺奶/奶泡点缀） | 一抹 | 60–80ml | 浓缩为主，奶只"点一下" |
| 可塔朵 Cortado | 1 份 | ≈ 等量温奶（约 40–60g） | 几乎无 | 90–120ml | 浓缩:奶 1:1，最平衡 |
| 澳白 Flat White | 1 份（多用 ristretto） | ≈ 100–150ml 微泡奶 | 极薄（≤0.5cm） | 150–160ml | 浓缩突出、丝绒感 |
| 卡布奇诺 Cappuccino | 1 份 | ≈ 等量热奶（约 60ml） | 厚（1–2cm） | 150–180ml | 1:1:1 三层，厚泡是灵魂 |
| 拿铁 Caffè Latte | 1 份 | 多奶（约 150–240ml） | 薄（0.5–1cm） | 220–300ml | 奶味主导、最顺 |
| 摩卡 Mocha | 1 份 | 热奶 + 巧克力酱 | 薄 + 巧克力 | 200–250ml | 拿铁加巧克力，甜 |
| 康宝蓝 Con Panna | 1 份 | — | — | 40–60ml | 浓缩 + 一坨鲜奶油 |
| 美式 Americano | 1 份 | —（加热水） | — | 150–250ml | 见一节「意式变体·美式」 |

> 表内"1 份浓缩"= 双份粉碗出 36–40g（约 1:2）；单份浓缩约 25–30g。各家杯子不同，比例是相对关系不是死数字。数据于 2026-07-15 联网核对（expertcafe / completehomebarista / coffeebros / coffee-guide.jp）。

### 冰手冲（Iced Pour-over）
- 器材/材料：滤杯（V60 等）+ 对应滤纸、手冲壶、冰杯（提前冰镇更佳）、冰、粉、热水。
- 粉量 15g / 冰约 100g / 热水总量约 150g（粉水比约 1:16–1:17，冰算部分水量）/ 水温 90–93℃（浅烘可高）/ 研磨中细。
- 步骤：粉入滤杯 → 先倒少量水闷蒸 30s → 分次注水到目标热水量 → 直接落在冰上冷却，或先萃热咖啡再倒冰上。
- 浅烘花香豆做冰手冲最出彩；深烘易显苦，少做冰手冲。

### 意式特调通用要点
- **萃取方案先定**：用 espresso（1:2）还是 ristretto（1:1–1:1.5，更短更浓）？澳白/flat white/dirty 多用 ristretto 或标准 espresso。
- **豆子选择**：奶基类（拿铁/澳白/dirty/卡布）常用**中深烘拼配**；想突出豆子本身风味（SOE 澳白、特调）用**浅烘 SOE**。
- **粉碗与滤纸**：注意粉碗标称容量（18g/20g/22g），别超装；不同咖啡师做特调用不同粉碗/滤纸，提前备好对应规格。

### 经典奶咖逐款做法（比例已联网核实）

#### 卡布奇诺 Cappuccino（1:1:1 三层）
- **比例**：浓缩 1 份（双份 36–40g）+ 蒸奶 ≈ 等量（约 60ml）+ 厚奶泡（1–2cm），总量 150–180ml，用 5–6oz 小杯。
- **豆子**：中深烘拼配。
- **步骤**：① 预热小杯；② 萃取 1 份浓缩入杯；③ 打奶——注气 3–5 秒（比拿铁更久、泡沫更厚），转漩涡加热到 60–65℃；④ 先倒奶液融合，奶泡用勺铺表面约 1–2cm；⑤ 可筛可可粉/肉桂粉。
- **口诀**：卡布=厚泡小杯；想更奶往拿铁靠，想更浓往澳白/可塔朵靠。
- **核实**：传统意式 1:1:1（espresso : steamed milk : foam），杯容 150–180ml、奶泡≥2cm（expertcafe / completehomebarista / coffee-guide.jp，2026-07-15）。

#### 拿铁 Caffè Latte
- **比例**：浓缩 1 份 + 多奶（约 150–240ml）+ 薄奶泡（0.5–1cm），总量 220–300ml，大杯。
- **步骤**：萃取浓缩 → 打奶（注气短、以漩涡加热为主，奶泡薄）→ 先倒奶、最后薄泡收尾 → 可拉花。
- **与卡布区别**：奶更多、杯更大、泡更薄；与澳白区别：奶更多。

#### 澳白 Flat White
- **比例**：浓缩 1 份（多用 ristretto 更浓）+ 微泡奶约 100–150ml，奶泡极薄（0.5cm 以下、与奶融合），总量 150–160ml。
- **步骤**：萃取 ristretto → 打"丝绒"微泡（几乎无独立泡沫层）→ 倒奶、表面是平的一层细泡。
- **注意**：澳白重点是"微泡融合"不是厚泡；别打成卡布。

#### 可塔朵 Cortado
- **比例**：浓缩 1 份 + 等量温热奶（约 40–60g），几乎无泡，总量 90–120ml。
- **豆子**：中深烘。
- **步骤**：萃取浓缩 → 倒等量微温/微泡奶（只加热、少注气）→ 融合即可。玻璃小杯。
- **特点**：平衡，奶只"降强度"不抢味。

#### 玛奇朵 Macchiato（两种做法相反，先问清）
- **Espresso Macchiato（意式玛奇朵）**：浓缩 1 份 + 一小勺奶/一抹奶泡"点"一下，总量 60–80ml。浓缩为主，奶只是标记。
- **Latte Macchiato（拿铁玛奇朵）**：反过来——先倒大量蒸奶（约 200ml）入高杯，再把 1 份浓缩缓缓注入中间，形成三层（奶/咖/奶），比例约 1:6，奶主导。
- **注意**：两种"玛奇朵"做法相反，给用户方案前先确认要哪种。

#### 摩卡 Mocha（拿铁 + 巧克力）
- **比例**：浓缩 1 份 + 巧克力酱 15–30g + 热奶约 60–80ml + 薄泡；顶部可加鲜奶油/巧克力碎。总量 200–250ml。
- **步骤**：杯底放巧克力酱 → 加热奶搅匀 → 萃取浓缩倒入 → 薄奶泡收尾 → 可选鲜奶油。
- **豆子**：中深烘拼配最搭巧克力。

#### 康宝蓝 Espresso Con Panna
- **比例**：浓缩 1 份（约 36g）+ 打发鲜奶油一坨（约 20–40ml）铺顶，总量 40–60ml。
- **步骤**：萃取浓缩到小杯 → 鲜奶油打至略流动浓稠 → 铺在咖啡表面 → 可筛可可粉。
- **注意**：鲜奶油别打太硬；部分国家把"维也纳咖啡"也叫 Con Panna，做的时候确认清楚。

#### 爱尔兰咖啡 Irish Coffee（含酒精甜饮）
- **材料**：热咖啡/双份浓缩+水（约 150ml）+ 爱尔兰威士忌 30–40ml + 红糖 1 茶匙 + 打发鲜奶油（浮顶层）。
- **步骤**：温杯 → 红糖 + 热咖啡搅匀 → 加威士忌 → 鲜奶油用勺背轻轻引流浮在表面（不搅拌）。
- **注意**：含酒精，属特调甜饮；想做无酒精版需先确认。

#### 维也纳咖啡 Vienna Coffee
- **比例**：浓缩 1–2 份 + 热水约 100ml（做成小美式）+ 大量打发鲜奶油铺顶 + 巧克力碎装饰。
- **步骤**：做小美式 → 倒杯 → 顶部挖两勺打发鲜奶油 → 撒巧克力碎。
- **区别**：康宝蓝是"浓缩+薄奶油"，维也纳是"咖啡+厚奶油顶+巧克力"，更像甜点。

### Affogato 阿芙佳朵（咖啡 + 冰淇淋，甜品级）
- **材料**：香草冰淇淋/意式 gelato 1–2 球（约 50–100g，越密实越好；太蓬松的便宜冰淇淋会秒化变汤）+ 现萃浓缩 1 份（25–40ml，双份更浓）。可选：ristretto（更甜更厚）、少许 Amaretto/Frangelico 利口酒。
- **步骤**：① 杯子先冷冻 10–15 分钟（延缓融化）；② 舀冰淇淋入杯；③ 立刻现萃浓缩、直接浇在冰淇淋上；④ 马上上桌配小勺，**不要搅拌**，先吃热冷对比的几秒。
- **豆子**：中深烘（巧克力/焦糖/坚果调性）最搭；浅烘尖酸会与香草冲突。
- **注意**：属甜品而非饮品；浓缩要现萃现浇，等太久冰淇淋成汤。
- **核实**：经典意式 1 球 gelato（50–100g）+ 1 份热浓缩（25–40ml），ristretto 更佳、立即享用（expertcafe / brewingcoffees / myreverbcoffee，2026-07-15）。

### 三个好上手创意示例（含器材）
- **生椰拿铁**：浓缩 36g + 冰椰子饮 150g + 牛奶 100g，搅匀加冰；器材：意式机、18g 粉碗、奶缸。
- **柠檬美式**：浓缩 36g + 气泡水 150g + 少许柠檬汁（先少放，尝了再补）+ 冰；器材：意式机、冰杯。
- **椰云拿铁**（瑞幸式）：浓缩 36g + 椰浆（厚椰乳）80–120g + 牛奶 80g 打发（用奶泡器或蒸汽）成"椰云"覆盖 + 浓缩浇顶；器材：意式机、奶泡器/打奶器、玻璃杯。

### 热奶泡 vs 冰奶泡
- **热奶泡**（拿铁/卡布/澳白基础）：
  - 冷牛奶 5–8℃ 起始，蒸汽头插入液面下 1–2cm
  - 先**嘶**（注气）3–5 秒，让体积膨胀
  - 再下沉**咝——**（旋涡）持续加热
  - 目标温度 55–65℃，超过 70℃ 蛋白变性味道发烫
  - 厚度：澳白薄奶泡（0.5–1cm 微泡）/ 卡布厚奶泡（1–2cm）/ 拿铁薄泡
- **冰奶泡 / 冰打奶**（iced latte / 冰拿铁）：
  - 先用少量冰块 + 牛奶 150ml 在奶缸里打 5–10 秒（手动打奶器/小奶泡机）
  - 倒进装满冰的杯子，再倒 espresso 36g
  - **不要用蒸汽**打冰的（量少温度立刻升上来，打不发）
  - 也可以**先热打再冰镇**：蒸汽打完立刻倒进装冰的杯子里，会变成绵密冷奶泡

### 拉花入门参数
- 奶泡厚度：薄奶泡（流动性好），倾倒时流速稳定
- 拉花缸口对准杯中心，倒入点距离液面 5–10cm（高一点拉出"穿透"形成图案）
- 起步练"心形"：从中心注入 → 摆动 → 一条线收尾
- 失败常见原因：奶泡太厚 / 浓缩油脂太薄 / 拉花缸角度不对 / 太紧张手抖

### 调整
太苦→少放咖啡/多放奶水或调粗研磨；太甜→糖浆减半；太酸→柠檬少放或换中深烘豆。

### 变压萃取提示
若用意式机带**变压功能**想试变压萃取：本文件给的是固定压力起步参数；变压曲线需**联网查该机型社区/论坛方案**（见 SKILL.md「变压萃取」章节）。

## 十、挂耳咖啡（Drip Bag）

适用：挂耳包 + 杯子 + 热水（无需磨豆机/器具，新手与办公室入门最常用）

- 粉量：约 10–12g（挂耳包通常已装好，无需自己称）
- 水量：150–180g（粉水比约 1:15）
- 水温：88–92℃（深烘 88℃，浅烘 92℃，别用刚烧开的沸水冲浅烘）
- 时间：闷蒸 30 秒 + 总萃取 2–3 分钟
- 步骤：
  1. 撕开挂耳，把两边"耳朵"挂杯沿上。
  2. 先倒少量水（约粉量 2 倍）闷蒸 30 秒，看粉鼓起来。
  3. 分 2 次慢慢注水到目标量，别冲太猛。
  4. **萃取完立刻取出挂耳**（别一直泡在杯里，会越泡越苦）。
- 调整：太苦→水凉一点 / 少泡一会 / 取出挂耳别拖延；太酸→水温高一点。浅烘花香挂耳别用沸水。

## 十一、虹吸壶（Syphon / 赛风）

适用：虹吸壶（上壶 + 下壶）+ 滤布/滤纸 + 加热源 + 磨豆机（爱好者进阶仪式感选择）

- 粉量：15–20g，中细研磨（像细砂糖）
- 水量：下壶装 240–300g
- 水温：92–94℃（下壶煮沸后自然降温）
- 时间：插上壶 → 搅拌 → 总约 1:30–2:00
- 步骤：
  1. 下壶加水，加热至沸腾。
  2. 插上壶（真空把水吸上去），倒粉、搅拌。
  3. 计时约 60 秒，期间可再轻搅一次。
  4. 关火，咖啡回流下壶，倒出即可。
- 调整：太苦→粉磨粗 / 时间短 / 别煮太久；太酸→水温高一点 / 多搅一会。注意火候，别烧干下壶。

## 十二、土耳其咖啡（Cezve / Ibrik）

适用：土耳其咖啡壶（Cezve）+ 炉灶 + 超细粉（像面粉，不能用普通磨豆机，需专用或现磨）

- 粉量：7–10g 超细粉 / 杯
- 水量：约 60–90g / 杯
- 糖：按口味（原味 / 少糖 / 多糖），传统可加一小撮豆蔻
- 步骤：
  1. 冷水 + 粉 + 糖入壶，小火加热。
  2. 表面起泡沫（"首层泡沫"）时离火稍息，让粉沉淀。
  3. 再加热起泡，重复 2–3 次。
  4. 倒入小杯，**连细粉一起倒**，静置后粉沉淀杯底，喝上层、别喝到底。
- 特点：不过滤、口感浓稠，有独特品鉴体系。
- 调整：太苦→少煮一会 / 粉别太细；有焦味→火别太大。注意：超细粉别用寻常滤杯，会堵。

## 十三、闪萃 / 日式冰冲（Flash Brew）

适用：手冲壶 + 滤杯 + 冰 + 磨豆机

- 区别于**冷萃**（12h+ 慢萃）与**冰手冲**（先热萃后加冰）：闪萃是"热萃取直接过冰"，快速得到冰咖啡，更好保留香气。
- 粉量：15g / 冰 100–150g（放分享壶或杯中）/ 热水 150g / 粉水比约 1:16–1:17（冰算部分水量）
- 水温：90–93℃ / 研磨中细
- 步骤：杯里放冰 → 滤杯置其上 → 按手冲方式热萃（闷蒸 + 分段注水）直接落在冰上 → 冰融化即冷却。
- 调整：同手冲——太苦粉磨粗，太酸粉磨细；想更浓减少冰量。浅烘果香豆做闪萃最出彩。

## 十四、越南咖啡（Vietnamese Phin / 滴漏壶）

适用：越南滴漏壶（phin filter，带压粉盘）+ 深烘咖啡粉 + 炼乳 + 热水（甜浓冰饮代表，做法独特）

- 粉量：15–20g，中粗研磨（比手冲略粗，接近粗砂糖偏细）
- 炼乳：15–25g（约 1–2 汤匙，铺在杯底）
- 水量：80–100g 热水，水温 90–95℃
- 时间：滴滤约 4–5 分钟
- 步骤：
  1. 杯底先放炼乳。
  2. phin 放上粉，用自带压粉盘轻轻压平，架在杯口。
  3. 先注少量热水润湿粉（闷蒸约 30 秒），再慢慢注满。
  4. 盖上盖，让水自然滴完（4–5 分钟）。
  5. 冰饮：杯里先放冰块，滴完搅匀；热饮：直接搅匀喝。
- 调整：太苦 → 粉磨粗一点 / 少放粉 / 炼乳多加一点；太淡 → 粉磨细一点 / 炼乳少一点。
- 注意：phin 孔小，粉太细会堵；深烘豆才够味，浅烘会尖酸；炼乳是灵魂，没有可用甜牛奶替代但风味差很多。

## 重要提醒

- 调整原则（一次只改一个变量、换豆先做对比杯等）见 SKILL.md「铁律」，本节不重复。
- 网上的"名家/博主/咖啡馆配方"是参考，请按自己的器具与口味微调，详见 SKILL.md 的联网检索流程与 [search-queries.md](search-queries.md)。
- 豆子越新鲜、研磨越接近冲煮时刻，风味越稳定。

- 特调没有标准答案，好喝就行，按自己舌头调。
- 变压萃取（pressure profiling）需联网查机型曲线，本文件的意式参数为固定压力起步值，详见 [pressure-profiles.md](pressure-profiles.md)。

## 季节与环境微调

本文件给的是标准室温（18–25℃）下的起步参数。极端气温时需微调：

| 环境 | 调整方向 |
|------|------|
| **夏季高温**（> 28℃） | 水温降 1–2℃；冷萃/冰滴缩短浸泡 1–2h；粉易受潮，磨前检查有无结块 |
| **冬季低温**（< 15℃） | 水温升 1–2℃；充分预热杯具/滤器/手柄；出液后尽快饮用 |
| **潮湿季节**（梅雨/回南天） | 粉易吸潮结块、研磨不均；少量多次购买、密封保存；必要时用 RDT（喷雾瓶轻喷 1–2 下水再磨） |
| **干燥季节** | 磨豆机静电增大，粉飞溅；用 RDT 缓解，或敲击接粉杯 |
| **高海拔**（> 2000m） | 水沸点降低，手冲水温需实测（沸水约 93–95℃），意式机温控可能偏移 |
