# 稳妥起步参数（新手可直接照做）

以下为"第一次做也不容易翻车"的起步方案，覆盖九种常见做法。资深用户可据此微调。
所有方案都假设使用新鲜咖啡豆（烘焙后 2–4 周内最佳）与可称重量的电子秤。

**选什么豆子、豆子怎么影响萃取（深烘/浅烘/处理法/新鲜度），见 `references/beans.md`。**

**通用口诀（记牢）**：
> 苦调粗，酸调细；淡了粉多水少，浓了粉少水多；水流快调细，水流慢调粗。

---

## 一、意式浓缩（Espresso）

适用：半自动咖啡机 + 磨豆机（能调粗细）

- 粉量：18 克（双份）
- 出液：约 36 克（比例约 1:2）
- 水温：92–94℃
- 时间：25–30 秒
- 研磨：中等偏细，以 25–30 秒出完 36 克为准
- 步骤：
  1. 称 18 克粉，倒进手柄，铺平。
  2. 压平（轻压即可，不用大力）。
  3. 开机，同时计时。
  4. 到 36 克左右停（约 27 秒）。
- 太苦 → 粉磨粗一点（**苦调粗**）；太酸 → 粉磨细一点（**酸调细**）

## 二、手冲（Pour-over）

适用：手冲壶 + 滤杯（如 V60）+ 磨豆机 + 秤

- 粉量：15 克
- 水量：240 克（比例约 1:16）
- 水温：90–92℃（深烘降到 88℃，浅烘可到 93℃）
- 研磨：中等（像粗砂糖）
- 总时长：2 分 30 秒 – 3 分 30 秒
- 步骤（分段注水）：
  1. 粉倒入滤杯，先倒 30 克水"醒一下"（闷蒸）30 秒。
  2. 分 2–3 次慢慢加水到 240 克，画圈浇在粉上、别冲到滤纸边。
  3. 等水滤完即可。
- 太苦 / 焦 → 粉磨粗一点（**苦调粗**）；太酸 / 寡淡 → 粉磨细一点（**酸调细**）

## 三、法压壶（French Press）

- 粉量：15 克，粗研磨（像海盐）
- 水量：250 克，水温 93℃
- 时间：浸泡 4 分钟，压下去，倒出
- 调整：太苦→粉磨更粗；太酸→粉磨细一点或延长浸泡。

## 四、爱乐压（AeroPress）

适用：爱乐压壶 + 磨豆机（最宽容、不易翻车，适合新手）

- 粉量：15–17 克
- 水量：220–250 克
- 水温：80–90℃（深烘用低一点，浅烘用高一点）
- 研磨：中细（比手冲细，比意式粗）
- 时间：浸泡约 1 分钟，再压约 20–30 秒
- 步骤：
  1. 装粉，倒水到目标量，用勺子搅两下。
  2. 闷 1 分钟。
  3. 放上滤盖，慢慢往下压（20–30 秒压完）。
- 调整：太苦→粉磨粗一点或水降温；太酸→粉磨细一点或水升温。

## 五、摩卡壶（Moka Pot）

适用：摩卡壶 + 炉灶（煤气/电磁炉均可）

- 粉量：18–20 克，中细研磨（像细砂糖偏细）
- 水：下壶装冷水，到安全阀下面一点点（别超过）
- 火：中小火
- 时间：上壶开始出咖啡约 1–2 分钟；听到"咕噜咕噜"声立刻离火
- 步骤：
  1. 下壶加冷水，粉碗装粉铺平（别压）。
  2. 拧紧上下壶。
  3. 中小火加热，听到出咖啡后转最小火收尾。
  4. 一听到"咕噜"声就关火、离炉。
- 调整：太苦/有焦味→火关小一点、粉磨粗一点、别煮到咕噜太久；有铁锈/生味→别让水烧干、粉别压太实。

## 六、冷萃（Cold Brew）

适用：任何容器 + 冰箱（夏天首选，几乎零失败）

- 粉量：50–70 克，粗研磨（像海盐）
- 水量：500–700 克，常温水或冷水
- 时间：冰箱浸泡 12–24 小时
- 步骤：
  1. 粉和水倒在一起搅匀。
  2. 盖好放冰箱。
  3. 时间到，用滤纸/滤网过滤即可喝（可加冰或水稀释）。
- 调整：太苦→粉磨更粗或时间缩短；太淡→粉多放点或时间拉长。

## 七、冰滴（Ice Drip / 冰酿）

适用：冰滴壶（或简单滴滤装置）

- 粉量：40–60 克，中粗研磨
- 冰水：400–600 克（冰 + 水混合）
- 时间：慢慢滴 4–8 小时（目标约每秒 1 滴）
- 步骤：
  1. 粉铺平在粉层。
  2. 上调阀门让冰水慢慢滴下（太快就调慢）。
  3. 接住咖啡，冷藏后喝。
- 调整：太快/太淡→把阀门调慢、粉磨细一点；太苦→粉磨粗一点、冰水比例多加水。

## 八、聪明杯（Clever Dripper）

适用：聪明杯 + 滤纸 + 磨豆机（像"泡着的手冲"，比手冲好控制）

- 粉量：15–20 克，中研磨（像粗砂糖）
- 水量：240–300 克，水温 90–93℃
- 时间：浸泡 2–3 分钟，再打开阀门滴下
- 步骤：
  1. 放滤纸，加粉，倒水到目标量。
  2. 盖好，闷 2–3 分钟。
  3. 把杯子放到杯子上，打开底部阀门，咖啡自己滴下。
- 调整：同手冲——太苦粉磨粗，太酸粉磨细。

## 九、咖啡特调与冰手冲（Signature / 创意饮 / Iced）

做特调或冰手冲前，**先列配方与器材清单**，再开始。下面给常见款的安全起步。

### 冰手冲（Iced Pour-over）
- 器材/材料：滤杯（V60 等）+ 对应滤纸、手冲壶、冰杯（提前冰镇更佳）、冰、粉、热水。
- 粉量 15g / 冰约 100g / 热水总量约 150g（粉水比约 1:16–1:17，冰算部分水量）/ 水温 90–93℃（浅烘可高）/ 研磨中细。
- 步骤：粉入滤杯 → 先倒少量水闷蒸 30s → 分次注水到目标热水量 → 直接落在冰上冷却，或先萃热咖啡再倒冰上。
- 浅烘花香豆做冰手冲最出彩；深烘易显苦，少做冰手冲。

### 意式特调通用要点
- **萃取方案先定**：用 espresso（1:2）还是 ristretto（1:1–1:1.5，更短更浓）？澳白/flat white/dirty 多用 ristretto 或标准 espresso。
- **豆子选择**：奶基类（拿铁/澳白/dirty/卡布）常用**中深烘拼配**；想突出豆子本身风味（SOE 澳白、特调）用**浅烘 SOE**。
- **奶泡与奶量**：
  - 澳白 / flat white：薄奶泡（约 0.5–1cm 微泡），牛奶约 100–150ml，浓缩 1 份（ristretto 更浓）。
  - 卡布奇诺：厚奶泡（约 1–2cm），牛奶约 100–150ml。
  - dirty：不打奶泡，冰牛奶/冰博克约 150ml 打底，浓缩（ristretto 更佳）直接浇上、不搅拌，喝时分层。
- **粉碗与滤纸**：注意粉碗标称容量（18g/20g/22g），别超装；不同咖啡师做特调用不同粉碗/滤纸，提前备好对应规格。

### 两个好上手示例（含器材）
- 生椰拿铁：浓缩 36g + 冰椰子饮 150g + 牛奶 100g，搅匀加冰；器材：意式机、18g 粉碗、奶缸。
- 柠檬美式：浓缩 36g + 气泡水 150g + 少许柠檬汁（先少放，尝了再补）+ 冰；器材：意式机、冰杯。

### 调整
太苦→少放咖啡/多放奶水或调粗研磨；太甜→糖浆减半；太酸→柠檬少放或换中深烘豆。

### 变压萃取提示
若用意式机带**变压功能**想试变压萃取：本文件给的是固定压力起步参数；变压曲线需**联网查该机型社区/论坛方案**（见 SKILL.md「变压萃取」章节）。

## 重要提醒

- 一次只改一个变量（研磨 / 水温 / 粉水比 / 时间 / 粉碗），方便判断是哪个动作起了作用。
- 网上的"名家/博主/咖啡馆配方"是参考，请按自己的器具与口味微调，详见 SKILL.md 的联网检索流程。
- 豆子越新鲜、研磨越接近冲煮时刻，风味越稳定。
- **粉碗与滤纸**：做意式前确认粉碗标称容量（18g/20g/22g），别超装；手冲前备好对应滤纸（V60 锥形/蛋糕形/金属），不同滤材影响流速与口感。
- 特调没有标准答案，好喝就行，按自己舌头调。
- 变压萃取（pressure profiling）需联网查机型曲线，本文件的意式参数为固定压力起步值，见 SKILL.md「变压萃取」章节。
