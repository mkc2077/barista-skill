## 单变量铁律提醒 / Iron rule
每次只改一个变量，喝一口再改下一个。新豆本身就是一个变量——先用本配方跑一次基线。
Change ONE variable at a time. Sip before next change. New beans are themselves a variable; brew a baseline first.

## 示例 / Example

### 手冲 V60 中烘耶加雪菲
- 器具: V60 + 锥形滤纸
- 冲煮法: Pour-over
- 豆: 埃塞俄比亚耶加雪菲（中浅烘 · 水洗 · 耶加）
- 档位: 进阶
- 来源: internal baseline（参考粕谷哲 4:6）

| Item | Value |
|---|---|
| 粉量 | 15g |
| 水量 | 240g |
| 粉水比 | 1:16 |
| 水温 | 92℃ |
| 研磨度 | 中细（白砂糖粗细） |
| 时间 | 2:30 |
| 手法 | 4:6 分段（粕谷哲） |

预期风味: 花香、柑橘酸、清甜余韵

变量调节:
- 偏酸: 升高水温 / 加粗研磨 / 加长闷蒸
- 偏苦: 降低水温 / 调细研磨 / 缩短接触
- 偏薄: 加粉量 / 调细研磨 / 改浅烘豆
