# 故障决策树：意式 / 手冲 / 磨豆机 / 奶泡

> **新手反馈味道问题时**：先按对应决策树问 1–2 个澄清问题，再给建议。避免"猜错方向→调错变量→越调越差"。
> **资深直接看底部"参数化诊断"表**。

---

## 一、意式味道问题

### 决策树

```
用户说"浓缩味道怪" / "不好喝"
│
├─ 1. 太苦 / 焦 / 中药味？
│  ├─ Q1: 出液时间多长？
│  │  ├─ < 20s → 萃取过快
│  │  │  ├─ 粉太粗 → 调细研磨
│  │  │  └─ 粉量不足 → 增粉（确认粉碗容量）
│  │  ├─ 20–30s → 进入 Q2
│  │  │  ├─ 水温 >94℃ → 降温 1–2℃
│  │  │  ├─ 深烘豆 + 偏粗研磨 → 调粗到更合适档位
│  │  │  └─ 粉太细 + 25s+ 满 → 调粗研磨
│  │  └─ > 35s → 萃取过慢
│  │     ├─ 粉太细 → 调粗研磨
│  │     ├─ 布粉不均/有通道 → 重新布粉/加 WDT
│  │     └─ 填压太实 → 轻压
│  └─ Q1b: 出液外观？喷射/飞溅 → 粉太细或有通道
│
├─ 2. 太酸 / 尖酸 / 青涩？
│  ├─ Q1: 出液时间？
│  │  ├─ < 20s → 粉太粗 → 调细
│  │  ├─ 20–30s → Q2
│  │  │  ├─ 水温 <92℃ → 升温 1–2℃
│  │  │  ├─ 浅烘豆 + 偏粗研磨 → 调细
│  │  │  └─ 浅烘 SOE + 25s+ → 调细 + 升温
│  │  └─ > 35s → 反常，可能是豆子问题或布粉不均
│  └─ Q1b: 豆子新鲜度？> 1 个月 → 老豆，磨细升温
│
├─ 3. 淡如水 / 没味道？
│  ├─ 出液很快 → 粉太粗或粉量不足
│  ├─ 出液慢但还是淡 → 粉量不足或水温低
│  └─ 出液正常但淡 → 检查豆子新鲜度
│
├─ 4. 太浓 / 厚重发闷？
│  ├─ 出液很慢 → 粉太细
│  └─ 出液正常但闷 → 粉量过多 + 粉水比太小
│
├─ 5. 涩口 / 干涩刮舌？
│  ├─ 出液很慢 + 研磨很细 → 调粗
│  ├─ 水温过高 → 降温
│  └─ 通道效应（出液忽快忽慢）→ 重新布粉 / 加 WDT / 换粉碗
│
├─ 6. 闻着香喝着没味？
│  ├─ 粉量不足 → 增粉
│  ├─ 研磨偏粗 → 调细
│  └─ 水温低 / 萃取不充分 → 升温或延长时间
│
├─ 7. 浓缩喷溅（"喷泉"）？
│  ├─ 粉太细 + 压力过载 → 调粗
│  ├─ 布粉不均/有通道 → 重新布粉 + WDT
│  └─ 粉碗堵塞（粉太细 + 压太实）→ 调粗 + 轻压
│
├─ 8. 前几滴正常，然后突然断流？
│  ├─ 粉太细 + 压太实 → 调粗 + 轻压
│  ├─ 粉碗堵塞/压粉时压偏了 → 重做
│  └─ 机器压力不足 → 检查机器（已越界，建议联系售后）
│
└─ 9. Crema 异常？
   ├─ Crema 太黑/很薄 → 豆子过老/太干 → 换新豆 / 增湿
   ├─ Crema 太浅/无 → 粉太粗 + 压力不足 → 调细
   └─ Crema 偏黄、油脂味重 → 深烘豆正常现象
```

### 给新手的固定话术

> "我们做一次诊断～你回忆一下：①这杯的出液用了多少秒（从开水到停机）？②是很快就流完（< 20 秒）、正常（25–30 秒）、还是磨磨蹭蹭（> 35 秒）？③咖啡表面喷不喷？"
> 拿到答案再给方向。

### 给资深的参数化诊断

| 现象 | 可能原因 | 调法（优先级排序） |
|------|------|------|
| 苦+焦+过萃 | 研磨过细 / 时间过长 / 水温过高 / 深烘豆 | 1. 调粗研磨 1 档 2. 降水温 1–2℃ 3. 缩短时间 5s 4. 确认豆子 |
| 酸+尖+欠萃 | 研磨过粗 / 时间过短 / 水温过低 / 浅烘豆 | 1. 调细研磨 1 档 2. 升水温 1–2℃ 3. 延长时间 5s 4. 确认豆子 |
| 淡+水感 | 粉量不足 / 研磨过粗 | 1. 增粉 1–2g 2. 调细研磨 |
| 涩+干 | 通道效应 / 过萃 / 水温过高 | 1. 重新布粉 + WDT 2. 调粗研磨 3. 降水温 4. 换粉碗 |
| 喷溅 | 粉过细 + 通道 | 1. 调粗 + 重新布粉 + WDT |
| Crema 异常 | 豆子过老 / 粉过粗 | 换新豆 / 调细 |

## 二、手冲味道问题

### 决策树

```
用户说"手冲不好喝"
│
├─ 1. 太苦 / 焦？
│  ├─ 总时长 > 3:30 → 萃取过久
│  │  ├─ 粉太细 → 调粗
│  │  ├─ 注水太慢 → 加快注水速度
│  │  └─ 滤纸堵塞 → 换滤纸/用蛋糕杯
│  ├─ 水温过高（>94℃）+ 浅烘也苦 → 降温到 88–90℃
│  └─ 深烘豆 + 正常参数 → 还苦，磨粗 + 降温
│
├─ 2. 太酸 / 尖？
│  ├─ 总时长 < 2:00 → 萃取不足
│  │  ├─ 粉太粗 → 调细
│  │  ├─ 注水太快 → 慢一点，画圈小
│  │  └─ 水温过低（< 88℃）→ 升到 90–93℃
│  ├─ 浅烘豆 + 正常参数 → 还酸，磨细 + 升温到 93–95℃
│  └─ 新豆（< 7 天）→ 养几天再喝
│
├─ 3. 淡如水 / 没味道？
│  ├─ 粉水比太大（如 1:18+）→ 调到 1:15–1:16
│  ├─ 粉太粗 → 调细
│  └─ 注水太少/太快 → 慢注、画圈
│
├─ 4. 闷 / 厚重发不出风味？
│  ├─ 粉水比太小（< 1:14）→ 调到 1:15–1:17
│  ├─ 粉太细 → 调粗
│  └─ 注水太慢 → 加快
│
├─ 5. 涩口？
│  ├─ 水温过高（> 95℃）→ 降到 90℃
│  ├─ 粉过细 → 调粗
│  └─ 注水冲击滤纸（绕圈碰到壁）→ 中心画圈
│
└─ 6. 流速异常？
   ├─ 流速太慢/堵住 → 调粗研磨
   ├─ 流速太快 → 调细研磨
   └─ 流速忽快忽慢 → 滤纸贴壁不好/布粉不均
```

### 给新手的固定话术

> "手冲做完后总时长大概多少？3 分钟内流完还是 3 分半还没流完？另外水温你用的多少度？"
> 答案通常能直接定位问题。

## 三、磨豆机问题

| 现象 | 原因 | 解决 |
|------|------|------|
| 出粉很慢 / 磨豆机很吃力 | 豆子太硬（深烘/陈豆）、刀盘钝 | 1. 调粗 1–2 档 2. 倒出少部分再磨 3. 考虑换豆或保养刀盘 |
| 静电大 / 粉飞溅 / 粉粘壁 | 干燥天气、咖啡粉太细 | 1. 用喷雾瓶在豆子上喷 1–2 下（"RDT"） 2. 调粗 1 档 3. 敲击接粉杯 |
| 粉结块 / 细粉过多 | 豆子含水率高、磨豆机校准问题 | 1. 养几天再用 2. 清理刀盘残留 3. 检查刻度环是否松动 |
| 出粉量与刻度不符 | 粉碗/容器残留、未清零 | 1. 每次接粉前清零 2. 用接粉杯 + 拍平 |
| 刻度盘松动 | 长期使用磨损 | 1. 联系品牌维修 2. 临时可用记号笔在刻度环上做标记 |
| 颗粒度不均（粗细混） | 刀盘钝 / 豆子问题 | 1. 换新豆测试 2. 联系售后 |

## 四、奶泡问题

| 现象 | 原因 | 解决 |
|------|------|------|
| 打不出泡 / 牛奶没变化 | 蒸汽压力不足 / 奶温太低 | 1. 检查机器是否出蒸汽（出口有"嘶嘶"声） 2. 奶温升到 5–10℃ 冷藏 3. 蒸汽头位置 |
| 奶泡太粗 / 像肥皂泡 | 蒸汽嘴位置太浅 | 蒸汽嘴插入液面下 1–2cm，靠"嘶嘶"声 |
| 奶泡太薄 / 流动性太强 | 蒸汽嘴插入太深 | 蒸汽嘴靠近液面，听到"咝——"的连续声 |
| 奶温上来但还没发泡就停了 | 蒸汽量不够 / 蒸汽头堵 | 1. 清洗蒸汽头 2. 排空蒸汽管道水分 3. 拔插重试 |
| 奶温过热（>70℃）/有焦味 | 打过头了 | 35–65℃ 是奶泡最佳温区，超 70℃ 蛋白变性、味道发烫 |
| 拉花不出图案 | 奶泡太厚 / 流动性差 | 1. 重新打奶泡（蒸汽嘴浅一点，让"嘶"声出现） 2. 拉花缸先润一遍 |
| 拉花颜色淡 / 纹路模糊 | 浓缩油脂太薄 / 拉花缸角度不对 | 1. 检查浓缩 crema 厚度 2. 拉花缸口对准杯中倒入点 |

## 五、其他常见故障

| 现象 | 可能原因 | 建议 |
|------|------|------|
| 意式机出咖啡像滴水 | 粉太细 + 压太实 + 压力不足 | 调粗 + 轻压；如仍不行，联系售后 |
| 手冲前 30 秒不出液 | 滤纸贴壁不好 / 粉太细 | 1. 用热水先润滤纸 2. 调粗研磨 |
| 法压压不下去 | 粉太细 | 调粗 1 档或换粗研磨 |
| 爱乐压压不出来 | 滤纸堵了 / 压太用力 | 换滤纸、慢慢压 |
| 摩卡壶喷出来 | 粉太细 / 压太实 / 火太大 | 1. 调粗 2. 粉别压 3. 火关小 |
| 冷萃有涩味 | 时间过长 / 粉太细 | 缩短时间（12h 而非 24h）+ 调粗 |

## 六、铁律提醒

- **一次只改一个变量**（研磨 / 水温 / 粉水比 / 时间 / 粉碗）。
- **改完喝一口再判断**，不要连续改 3 个变量然后问"为什么还是不对"。
- **记笔记**：建议用户用手机备忘录记录每次改的参数和结果，下一杯就能精准定位。
- **换豆先按旧参数做一杯**：确认是豆子问题再调（避免"豆子 + 参数同时变"造成混乱）。
